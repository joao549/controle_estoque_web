Programa para gerenciar produtos. 

Tela de login (admin/admin) - Por questões de segurança este será o único login e caso necessário outro, falar diretamente com o desenvolvedor.

tela de visualizar - produtos gera uma lista com todos os produtos inseridos e funções de detalhes, excluir ou editar

Botão de Adicionar - o redirecionará para outra tela onde poderá ser adicionado um novo produto. Exige que seja informado o valor e nome do produto

Editar - permite que o que foi adicionado anteriormente seja alterado.

Excluir - Excuirá o produto do banco de dados
